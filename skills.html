Minhas habilidades

Minhas habilidade incluem pacote Adobe, Photoshop intermediário para manipulações simples e desenvolvimento de esboços. Illustrator intermediário para criação, manipulação de vetores e ícones. Indesign para diagramação de diversos matérias, como revistas e folhetos.

Relacionadas ao mundo da tecnologia, possuo conhecimentos básicos em HTML, CSS e Java Script, utilizados no desenvolvimento desta página. Conhecimentos em lógica de programação para resolução de problemas.
